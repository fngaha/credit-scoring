# Description du jeu de données

- Source : OpenML (dataset `credit-g`)
- Observations : 1000
- Variables : 21 (20 explicatives + 1 cible)
- Cible : `class` ("good" = bon payeur, "bad" = risque élevé)
- Types : 7 numériques, 13 catégorielles

## Objectif
Comprendre la composition et le rôle de chaque variable dans la prédiction du risque de crédit.
